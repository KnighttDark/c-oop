# Relationships

- [ ] Aggregation: Mối quan hệ giữa các lớp, một lớp chứa các lớp khác
- [ ] Composition: Mối quan hệ giữa các lớp, một lớp chứa các lớp khác và không thể tồn tại nếu lớp chứa bị hủy
